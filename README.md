# 🤖 LangChain HuggingFace Chatbot

A simple AI chatbot built with **Streamlit**, **LangChain**, and **HuggingFace Transformers**.  
This app allows users to ask questions in a chatbot interface and receive AI-generated responses.

---

## 🚀 Features

- Interactive chatbot UI with Streamlit
- Uses a free HuggingFace model
- LangChain prompt handling
- Maintains chat history
- Simple and beginner-friendly AI project

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- HuggingFace Transformers
- PyTorch
